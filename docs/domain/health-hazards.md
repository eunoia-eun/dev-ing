# 유해인자별 건강장해 데이터

## 원천

- **근로자건강진단 실무지침 제3권 「유해인자별 건강장해」** (산업안전보건연구원, 2024 — [2024-산업안전보건연구원-784])
- 저장소 루트의 PDF(729쪽)에서 **파싱**하여 생성했습니다.
- 구조화 결과: `src/infrastructure/seed/hazardHealthDetails.json`

## 매핑

문서는 [별표 22]와 **동일한 분류·순서**(유기화합물-1 가솔린, …)로 구성되어, 카탈로그와 1:1 매핑됩니다.
키는 `"<분류코드>-<번호>"` (예: `CHEM_ORGANIC-41` = 벤젠).

| 분류(문서 표기) | 코드 | 수록 |
| --- | --- | --- |
| 유기화합물 | CHEM_ORGANIC | 109 |
| 금속류 | CHEM_METAL | 20 |
| 산 및 알칼리류 | CHEM_ACID_ALKALI | 8 |
| 가스상 물질류 | CHEM_GAS | 14 |
| 허가대상물질 | CHEM_PERMIT | 12 |
| 금속가공유 | CHEM_MWF | 1 |
| 분진 | DUST | 7 |
| 물리적 인자 | PHYSICAL | 5 (소음·진동·방사선·고기압·저기압) |
| 야간작업 | NIGHT_WORK | 2 |

→ **181종 중 178종**에 건강장해 상세가 매핑됩니다.
미수록 3종은 **유해광선(자외선·적외선·마이크로파/라디오파, PHYSICAL 6·7·8)** 으로 원문에 별도 섹션이 없습니다. UI는 이 경우 "자료 없음"을 표시합니다.

## 데이터 형태 (`HazardHealthDetail`)

```jsonc
{
  "docName": "벤젠",
  "cas": "71-43-2",
  "synonyms": "벤졸, …",                       // 1. 동의어
  "physicalChemical": "· CAS No 71-43-2\n· 분자량 …", // 2. 물리·화학적 성질
  "exposureLimitKr": {                              // 한국 고용노동부 노출기준(2020) — 항목별 구조화
    "twa": "0.5", "stel": "2.5", "carcinogen": "1A", "mutagen": "1B"
  },
  "absorption": "∙ 흡수 : …\n∙ 대사 : …",        // 5. 흡수·대사·배설·반감기
  "targetOrgans": ["신경계", "간담도계", "신장", "피부", "조혈기계"],
  "uses": "다른 유기화학물의 중간산물, …",        // 3. 발생원 및 용도
  "process": "주로 화학, 페인트, 고무, 인쇄 …",   // 4. 노출 공정
  "healthEffects": "(1) 급성 건강영향 … (2) 만성 … (3) 발암성 …" // 6. 표적장기별 건강장해
}
```

- 본문 섹션 1·2·3·4·5·6을 각각 추출하고, 각주 위첨자·페이지 머리글을 제거해 정리했습니다(필드별 길이 상한 적용).
- `targetOrgans`는 만성 영향의 "장기별 :" 항목에서 추출했습니다.
- **`exposureLimitKr`**(`KoreanExposureLimit`): 물질 본문이 아니라 앞쪽 표 **「한국 고용노동부 화학물질 및 물리적 인자 노출 기준표(2020.1.14.)」** 에서 CAS로 매칭(178종 중 145종, 분진·물리·일부 허가물질은 표에 없어 미매칭).
  - 표의 **탭 칼럼**(TWA-ppm/㎎㎥, STEL-ppm/㎎㎥, 피부흡수, 발암성)을 토큰 유형으로 분류해 `{twa, stel, skin, carcinogen, mutagen, reproToxic}` 로 구조화.
  - **단위(ppm/㎎㎥)**: 표의 dash(`-`) 위치로 칼럼을 복원하는 하이브리드 규칙으로 결정한다.
    - dash가 있는 행 → 칼럼 위치로 ppm/㎎㎥를 정확히 부여(예: 황산 `- 0.2 - 0.6` → 0.2·0.6 ㎎/㎥, 질산 `2 4 - -` → TWA 2 ppm).
    - dash가 없는 행 → 숫자 순서대로 TWA·STEL, 단위는 분류 기본값(금속·분진·금속가공유=㎎/㎥, 그 외=ppm).
    - 한계: 고체 유기물 일부는 원문에서 선두 dash가 누락돼 ppm으로 잘못 표기될 수 있음(예: 디니트로톨루엔). UI 각주에서 "단위는 고시 확인 권장"으로 안내.
  - `twa`의 `C` 접두는 최고노출기준(Ceiling). 발암성/변이원성/생식독성 등급은 1A·1B·2.

## 로딩 방식 (지연 로딩)

데이터(약 1MB)는 초기 번들을 무겁게 하지 않도록 **동적 import**(`StaticHazardHealthProvider`)로
카탈로그 상세를 처음 열 때만 별도 청크로 로딩됩니다.

```
UI(CatalogPage) → hazard.getHealthDetail(ref)
  → HazardHealthDetailProvider(port)
  → StaticHazardHealthProvider: await import('hazardHealthDetails.json')  // 1회 캐시
```

## 재생성 방법

원문 PDF에서 다시 만들려면, 텍스트를 추출(`pdf-parse`)한 뒤 분류헤더(`유기화합물-N` 등)로
섹션을 나누고 「표적장기별 건강장해」 구간을 잘라 키 `CODE-no`로 매핑합니다.
(작업 시 사용한 파서 로직은 커밋 메시지/PR 설명을 참고. 결정적 변환이므로 동일 입력→동일 출력.)
